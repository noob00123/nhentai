# hs-nhentai
Scraping/downloading [nhentai](https://nhentai.net) galleries and JSONs.

## Usage

### Help
```bash
$ nhentai --help
```

## Installation

### Linux

```bash
$ https://github.com/tensorknower69/nhentai
$ cd nhentai
$ stack install
```

## Uninstallation

### Linux
```bash
$ rm ~/.local/bin/nhentai
```
